## 🚀 STEP-BY-STEP DEPLOYMENT GUIDE

### ─────────────────────────────────────────
### STEP 1 — Install Python on your laptop
### ─────────────────────────────────────────
1. Go to: https://python.org/downloads
2. Download Python 3.11 or higher
3. During install → ✅ CHECK "Add Python to PATH"
4. Verify:
```
python --version
```
Expected: `Python 3.11.x`

---

### ─────────────────────────────────────────
### STEP 2 — Install laptop packages (one time)
### ─────────────────────────────────────────
Open Command Prompt / Terminal and run:
```
pip install google-api-python-client google-auth-oauthlib
```
(Only these 2 are needed on your laptop. Everything else runs on GitHub cloud.)

---

### ─────────────────────────────────────────
### STEP 3 — Get Groq API Key (2 min)
### ─────────────────────────────────────────
1. Go to: https://console.groq.com
2. Click "Sign Up" → free account
3. Go to "API Keys" in left sidebar
4. Click "Create API Key"
5. Copy the key — it starts with `gsk_...`
6. Save it somewhere safe (you'll paste it in GitHub Secrets later)

---

### ─────────────────────────────────────────
### STEP 4 — Get Pexels API Key (2 min)
### ─────────────────────────────────────────
1. Go to: https://www.pexels.com/api/
2. Click "Get Started" → create free account
3. Your API key is shown on the dashboard immediately
4. Copy and save it

---

### ─────────────────────────────────────────
### STEP 5 — Set Up YouTube API (5 min)
### ─────────────────────────────────────────
1. Go to: https://console.cloud.google.com
2. Sign in with the Google account that has your YouTube channel
3. Click "Select a Project" at top → "New Project"
4. Name it: `DarkMindVault` → Click "Create"
5. In the left menu: "APIs & Services" → "Enable APIs & Services"
6. Search: `YouTube Data API v3` → Click it → Click "Enable"
7. Go to "APIs & Services" → "Credentials"
8. Click "Create Credentials" → "OAuth 2.0 Client ID"
9. If asked to configure consent screen:
   - User Type: External → Create
   - App name: DarkMindVault
   - Support email: your email
   - Click "Save and Continue" through all steps
10. Back in Credentials → "Create Credentials" → "OAuth 2.0 Client ID"
11. Application type: "Desktop App"
12. Name: "DarkMindVault" → Click "Create"
13. Click the ⬇️ download button next to your new credential
14. A JSON file downloads — rename it to: `client_secrets.json`
15. Place `client_secrets.json` in the same folder as `auth_setup.py`

---

### ─────────────────────────────────────────
### STEP 6 — Connect Your YouTube Channel (1 min)
### ─────────────────────────────────────────
Open Terminal in your project folder and run:
```
python src/auth_setup.py
```

What happens:
1. Your browser opens automatically
2. Google login page appears
3. Log in with your YouTube channel's account
4. Click "Allow" to grant upload permission
5. Browser shows "Authentication successful"

Two files are automatically created:
- `YOUTUBE_CREDS_B64.txt`
- `YOUTUBE_TOKEN_B64.txt`

⚠️ IMPORTANT: Keep these files private. Never share them or commit them to GitHub.

## ❓ TROUBLESHOOTING

| Problem | Cause | Fix |
|---------|-------|-----|
| Pipeline fails at script step | GROQ_API_KEY wrong | Check secret has no extra spaces |
| Videos not uploading | Token expired | Re-run auth_setup.py, update secret |
| No footage in video | PEXELS_API_KEY wrong | Check secret, system uses gradient fallback |
| Actions not running | Workflows disabled | Go to Actions tab → Enable |
| "quota exceeded" YouTube | >6 uploads/day | Default is 6/day, within free quota |
| Action timeout (>3 hours) | Server too slow | Reduce ACTIVE_LANGUAGES in config.py |
